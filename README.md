Urban Performance Club
A responsive gym website built with HTML5, CSS3, and Bootstrap 5. Features strength
training, conditioning programs, and a dedicated boxing gym in Birmingham’s Jewellery
Quarter.
View Live Site
About This Project
Urban Performance Club is a comprehensive, mobile-first website created for the Code
Institute Level 5 Diploma in Web Application Development. The project demonstrates
advanced frontend skills by building a professional gym website with interactive
components and enhanced user experience.
Key Features:
4 fully responsive pages (Home, About, Gallery, Sign Up)
Bootstrap 5 integration (carousel, accordion, tables, forms)
Interactive FAQ and testimonial sections
Enhanced signup form with demographic fields
Mobile-first design optimized for all devices

Deployment
GitHub Pages:
1. Go to repository Settings → Pages
2. Source: Deploy from branch → main → / (root)
3. Save and wait 2-3 minutes
4. Live site: https://hassib-95.github.io/urban-performance-club/
Local Development:
git clone https://github.com/hassib-95/urban-performance-club.git
cd urban-performance-club
# Open index.html in browser
